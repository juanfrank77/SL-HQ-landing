6. Code Example

Here's the CSS that makes it all work:

/* Global Focus Styles */ *:focus { outline: 3px solid var(--primary-blue); outline-offset: 3px; } *:focus:not(:focus-visible) { outline: none; } *:focus-visible { outline: 3px solid var(--primary-blue); outline-offset: 3px; } /* Button Focus on Dark Backgrounds */ .btn-primary:focus-visible { outline: 3px solid #ffffff; outline-offset: 4px; }
